# Institute of Digital Risk (IDR) - Landing Page 🌐

A sleek, modern, and highly interactive landing page for the **Institute of Digital Risk (IDR)**, an industry-led training and deployment institute for digital, cyber, and AI risk practitioners. 

Built entirely with Vanilla web technologies (HTML, CSS, JavaScript) without the use of any external frameworks or libraries.

### 🔗 Live Demo
**[View the Live Deployment Here](https://prashanthdudekula.github.io/-Institute-of-Digital-Risk-IDR-/)**

---

## ✨ Features

* **Modern Dark UI:** A striking dark theme with vibrant orange accents (`#FF6B1A`) that conveys authority, urgency, and precision.
* **Fully Responsive:** Fluid layouts built with CSS Grid and Flexbox, ensuring a seamless experience across desktop, tablet, and mobile devices.
* **3D CSS Animations:** Features a rotating, isometric 3D cube built entirely with CSS transforms to symbolize multi-dimensional risk management.
* **Canvas Particle Network:** A dynamic, interactive background using the HTML5 `<canvas>` API to draw connecting nodes.
* **Intersection Observer Animations:** Smooth scroll-reveal effects and animated statistical counters that trigger as elements enter the viewport.
* **Infinite Ticker Tape:** A continuously scrolling CSS animation highlighting key industry terms (e.g., AI Governance, NIST CSF, ISO 27001).
* **Interactive Mobile Navigation:** A custom hamburger menu that opens a full-screen blurred drawer for smaller viewports.
* **Contact Form UI:** A beautifully styled, fully responsive contact form with a simulated success state upon submission.

---

## 🛠️ Technologies Used

* **HTML5:** Semantic markup (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`).
* **CSS3:** Custom variables (CSS properties), CSS Grid, Flexbox, keyframe animations, 3D transforms, and media queries.
* **Vanilla JavaScript (ES6+):** DOM manipulation, Event Listeners, `IntersectionObserver` API, and `requestAnimationFrame` for smooth canvas rendering.
* **Fonts:** Google Fonts (*Bebas Neue* for display, *Outfit* for body, *JetBrains Mono* for technical accents).
